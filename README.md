#### 📊 PhonePe Transaction Analysis Dashboard
## Project Overview

This project analyzes digital payment transactions using Power BI to uncover customer behavior, transaction trends, payment success rates, and service usage patterns.

The dashboard provides actionable business insights through interactive visualizations and KPI tracking, enabling stakeholders to monitor digital payment performance effectively.

## Key Insights

✔ Platform maintains a 96.15% payment success rate

✔ Digital payment usage is highest during weekdays.

✔ Monthly transaction values remain relatively stable with moderate fluctuations.

✔ Loan-related services generate one of the highest transaction values.

✔ User distribution is balanced across different age groups.

✔ Failed transactions represent only a small proportion of total transactions.
